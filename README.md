## Open Conversation Button for Thunderbird

A Thunderbird addon that adds a toolbar button to open selected email's conversation.

### Screenshot

![@screenshot](screenshot.png)

### Installation

See `dist/` folder or visit [Mozilla addon site][amo].

### Building

    npm install
    grunt build-xpi
